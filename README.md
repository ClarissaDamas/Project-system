○ Como rodar:
Clonar o repositório
Criar um ambiente virual
pip install -r requirements.txt.
Adicionar o arquivo .env + SECRET_KEY.
python manage.py runserver para acessar o servidor


○ Como criar DB/migrations:
python manage.py makemigrations
python manage.py migrate
Para acessar o painel de controle usar ---> python manage.py createsuperuser
